Array 연산

  • Array연산은 기본적으로 크기가 서로 동일해야 진행된다.
  • 이때 같은 위치에 있는 요소들 끼리 연산된다.

       [4, 5, 6]])

       [13, 14, 15]])

Array boolean 인덱싱(마스크)

  • Array의 다차원 인덱싱을 응용하여 boolean 인덱싱을 할 수 있다.
  • 이러한 방법을 마스크라고 이야기한다
  • 마스크처럼 원하는 요소만을 꺼낼 수 있다.
array(['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H'], dtype='<U1')
