Python 문법 7

파이썬 클래스(Class)

  • 클래스는 데이터와 기능(함수)을 묶어 놓은 객체이다.

Class 기본 형식

  • 어트리뷰트(attributes)란?
    • 클래스 내부에 포함되어있는 함수(메서드)나 변수등을 의미한다.
  • 인스턴스(instance)객체
    • 인스턴스란 클래스에 의해 만들어진 객체이다.
    • 인스터스가 가능한 유일한 연산은 어트리뷰트 참조 이다.
  • 메서드(method)객체
    • 객체에 포함되어 있는 함수를 의미한다.
    • 클래스에서의 메서드 객체는 사용자가 정의할 수 있는 객체가 있는 반면, 몇 가지 특수한 역할을 하는 메서드도 존재한다.

클래스 상속

  • 상속이란?

    • 물려주는 클래스(Parent Class, Super class)의 내용(속성과 메소드)을 물려받는 클래스(Child class, sub class)가 가지게 되는 것입니다.
    • EX) 국가 클래스(Parent) 한국, 중국, 일본, 미국 클래스(Child)

  • 상속 예시

  • Person클래스(부모) Korean클래스(자식)
  • 두 클래스 모두 whoAmI() 메서드가 존재한다. 이 둘의 각각 값은 다르다.
    • 그렇기 때문에 출력에서 동일한 메서드를 호출했지만, 다른 값이 출력된다.
  • 이처럼 상속은 같은 메서드라도 자식 객체에서 다르게 정의하여 사용 할 수 있다.
  • 또한, 부모클래스에서 정의한 메서드를 자식 클래스에서 사용하는 것이 가능하며 거기에 자식 클래스에서 정의한 메서드 또한 사용이 가능하다
  • 자식클래스 사용 = (부모클래스(메서드1, 메서드2, 메서드3)+자식클래스(메서드4, 메서드5))

파이썬 캡슐화(Encapsulation)

  • 클래스를 정의할 때 내부의 속성과 메서드를 묶어서 하나의 단위로 처리할 수 있다.
  • 캡슐화 예시
  • 맨처음 tv.sell()은 클래스에서 가장 처음 정의된 값인 500이 출력되었다.
  • 두번쨰 tv.sell()에서 가격을 1000으로 바꿔 출력하려 했지만 바뀌지 않았다.
    • 이는 클래스 내부에 _가 있는 속성이나 메서드는 관행적으로 private으로 약속하고 처리되기 떄문이다.
  • 이를 하기 위해선 추가 함수를 작성해 캡슐을 포맷해주는 과정을 거쳐야한다.
    • 포맷 이후, 마지막 출력에서 300으로 바뀐것을 확인 할 수 있다.
