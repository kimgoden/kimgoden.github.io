Python 문법 9

파이썬 상속(Inheritance)

1. 기본적인 클래스

  • 어떤 회사에서 이름과 급여를 관리할 떄 위와 같은 방식으로 한다 가정해보자.
  • 한 직원당 이름과 급여를 모두 한땀한땀 입력해줘야만 한다.
  • 이름은 고유값이지만, 급여는 일정한 기준이 있다.
  • 이와 같은 경우 조건문(if)를 이용해 클래스를 작성 할 수 있다.

2. 조건문을 이용한 자동화 클래스

  • 위처럼 DEFAULT_TIP = 2000 으로 기본급을 선언한다면, emp1.tip 을 print 했을 때 자동으로 디폴트값이 출력된다.
  • 그러나 emp2의 경우 처럼 금액을 직접 선언해준다면? ==> 그 금액이 출력된다.
  • 하지만 여기서 MAX_TIP = 10000 즉, 최대 급여를 선언해주고 조건을 건다면, 얼마를 입력하든 MAX_TIP을 넘지 못한다.

3. 별도의 데이터를 이용한 클래스 메서드

  • 하지면 여전히 이름은 하나하나 다 설정해줘야 한다.
  • 이러한 이름을 별도의 파일(employee.txt)로 만들어 이용해본다.

4. 정적 메소드와 클래스 메소드의 차이

  • 정적 메소드(staticmethod)와 클래스 메소드(classmethod)는 인스턴스를 생성하고 호출하는 방식이 아니라, 클래스에서 바로 호출할 수 있습니다.
  • 정적 메소드와 클래스 메소드는 위의 코드와 같이 메소드 위에 @staticmethod , @classmethod를 붙여 주어야 합니다.
  • 정적 메소드는 self를 사용할 필요가 없으며 클래스 메소드는 cls를 사용하여 클래스 속성을 인자로 전달 받습니다.
  • 클래스 메소드는 cls로 현재 자기가 실행된 클래스 속성을 가져오고, 정적 메소드는 부모 클래스의 속성을 가져오게 됩니다.

5. 모방

  • 부모 클래스를 자식이 모방하는것.
  • 자식클래스가 부모 클래스를 참조하는 것을 의미한다.
